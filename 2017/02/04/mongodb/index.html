criteria的规则

假设某集合中有一个year域表示年份,country域表示国家，以此为例：
相等条件：find({"year": 2015,"country":"中国"})，查找2015年的中国的数据

对于非相等条件，mongodb采用条件操作符的形式表示，全部的条件操作符见 官方文档，以下总结一些常用的条件过滤：

  • “$lt”：小于
  • “$lte”：小于等于
  • “$gt”：大于
  • “$gte”: 大于等于
  • “$ne”：不等于
  • “$regex”: 正则匹配

filterDisplay的规则

filterDisplay限制输出的域的情况，默认输出全部域。

  • find(criteria,{<field1>:1,<field2>:1})：返回<field1>和<field1>以及默认的_id字段。
  • find(criteria,{<field1>:0,<field2>:0})：返回除了<field1>和<field1>的其他字段。

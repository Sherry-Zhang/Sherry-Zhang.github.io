scrapy学习(一) ：环境配置

发表于 2017-01-26

scrapy框架需要在Python 2.7，lxml，OpenSSL的基础之上创建。

系统：ubuntu 16.04 LTS

安装缺失的依赖包

ubuntu16.04中直接安装scrapy会报错，需要先手动安装一些缺失的依赖包

$ sudo apt-get install libxml2-dev libxslt1-dev python-dev libssl-dev libffi-dev  build-essential

安装lxml库

$ pip install lxml

安装openssl库

无须安装，ubuntu16.04中默认配置中有openssl，安装scrapy时会自动安装依赖。

安装scrapy

$ pip install scrapy

安装完成，检查

在命令行中输入scrapy，显示如下界面，表示安装成功。

(env0) zs@ubuntu:~/workspace/env0$ scrapy
